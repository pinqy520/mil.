# Mil.

> typescript版冲顶大会、百万xx等辅助答题程序（还在开发中，预计今晚搞定吧）


1. 安装tesseract以及语言包

``` 
$ brew install tesseract --with-all-languages
```

2. 初始化

```
$ git clone git@github.com:pinqy520/mil..git
$ cd mil.
$ yarn
```

3. 测试

```
$ yarn once
```
